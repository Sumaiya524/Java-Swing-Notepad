# Java-Swing-Notepad
Welcome to Java Swing Notepad, a simple yet powerful text editor designed for quick note-taking and document management. This application mimics the core functionalities of a traditional notepad with a clean user interface.A feature-rich desktop text editor built with Java Swing that supports file management, editing, sharing, and printing operations.
